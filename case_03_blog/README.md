# Case 03. Web-приложение для блога

Учебный проект для производственной практики: блог на FastAPI с HTML/CSS/JS frontend и PostgreSQL.

## Возможности

- регистрация и вход пользователя;
- создание, просмотр, редактирование и удаление постов;
- публичные посты и приватные посты с режимом "только по запросу";
- отправка и одобрение запросов доступа к скрытым постам;
- подписка на других пользователей;
- лента постов на основе подписок;
- теги и сортировка/фильтрация постов по тегам;
- комментарии к постам.

## Стек

- Python 3.12
- FastAPI
- PostgreSQL
- SQLAlchemy
- HTML/CSS/JavaScript
- Docker Compose

## Запуск

```bash
docker compose up --build
```

После запуска сайт доступен по адресу:

```text
http://localhost:8003
```

Интерактивная документация API:

```text
http://localhost:8003/docs
```
