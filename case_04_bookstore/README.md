# Case 04. Web-версия книжного магазина

Папка для второго практического проекта.

Планируемый стек: FastAPI, PostgreSQL, SQLAlchemy, HTML/CSS/JS, Docker Compose.

## Запуск

```bash
docker compose up --build
```

После запуска сайт доступен по адресу:

```text
http://localhost:8004
```

## Демо-данные

После запуска контейнеров можно заполнить базу тестовыми данными:

```bash
docker compose exec web python -m app.seed
```

Тестовые пользователи:

- `admin` / `admin123`
- `reader` / `reader123`

Ключевые функции:

- роли администратора и пользователя;
- каталог любимых книг;
- просмотр книги из библиотеки;
- сортировка по категории, автору и году написания;
- покупка или аренда на 2 недели, месяц или 3 месяца;
- управление книгами, ценами, статусами и доступностью;
- автоматические напоминания об окончании аренды.
