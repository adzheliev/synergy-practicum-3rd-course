# Case 05. Приложение "Дневник путешествий"

Папка для третьего практического проекта.

Планируемый стек: FastAPI, PostgreSQL, SQLAlchemy, HTML/CSS/JS, Docker Compose.

## Запуск

```bash
docker compose up --build
```

После запуска сайт доступен по адресу:

```text
http://localhost:8005
```

## Демо-данные

После запуска контейнеров можно заполнить базу тестовыми данными:

```bash
docker compose exec web python -m app.seed
```

Тестовые пользователи:

- `traveler` / `traveler123`
- `friend` / `friend123`

Ключевые функции:

- регистрация пользователя;
- добавление записей о путешествии;
- просмотр путешествий других пользователей;
- местоположение с геопозицией;
- изображения мест;
- стоимость путешествия;
- оценки удобства передвижения, безопасности и других параметров.
